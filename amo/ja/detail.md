# Summary

Firefox の右クリックメニューから重複するタブを閉じます。

# Description

ClickTabUniq は、Firefox の右クリックメニューから重複するタブを直接閉じるための拡張機能です。

重複判定の方法を選べます。

- URL
- ハッシュを除く URL
- タイトル

判定方法ごとに、削除対象の範囲を選べます。

- クリックした階層
- 階層ごと
- 全てのタブ

メニュー項目には、実行前に閉じられる重複タブ数が表示されます。

階層は、トップレベルのタブと各タブグループです。
ピン留めされたタブはトップレベルのタブとして扱われ、どの重複タブを残すかを決めるときは通常のタブより優先されます。
グループ内のタブを右クリックした場合は、トップレベルだけを対象にした削除範囲も表示できます。
このトップレベル範囲を選ぶとトップレベルのタブだけを判定するため、グループ内の重複はグループ範囲を選ばない限り残ります。
分割ビュー内のタブも、重複判定と削除の対象に含まれます。

通知を有効にすると、処理の進行状況、完了結果、階層ごとに閉じたタブ数を表示できます。

## プライバシー

ClickTabUniq は、重複タブを検出して閉じるためにタブへのアクセス権限を使用します。
閲覧データを収集または送信することはありません。
