## Summary

タブを増やし過ぎて同じページを何度も開いてしまっていても、何件閉じるか事前に確認しながら、簡単に重複タブを閉じられます。

## Description

ClickTabUniq は、同じページが複数のタブで開いているときに、Firefox の右クリックメニューから重複タブを閉じるための拡張機能です。実行前に閉じられるタブ数をメニューで確認できるため、影響を把握してから実行できます。

重複判定方法を選べます。

- URL
- ハッシュを除く URL
- タイトル

さらに、重複判定方法ごとに、削除範囲を選べます。

- クリックした階層
- 階層ごと
- 全てのタブ

階層とは、トップレベルのタブと各タブグループを指します。
ピン留めされたタブはトップレベルのタブとして扱われ、重複時は通常タブより優先して残ります。
トップレベルの重複削除ではトップレベルのタブだけが判定対象になり、グループ内の重複はそのまま残ります。
グループ内のタブを右クリックした場合、グループ内だけの重複削除もできます。
分割ビュー内のタブも重複判定と削除の対象です。

重複判定方法や削除範囲が1つしかない場合、右クリックメニューが短絡され、より少ないステップで実行できるようになります。

通知を有効にすると、処理の進行状況、完了結果、階層ごとに閉じたタブ数を表示できます。

## プライバシー

ClickTabUniq は、タブの URL とタイトルを比較するためにタブへのアクセス権限を使用します。閲覧データを収集または送信することはありません。
